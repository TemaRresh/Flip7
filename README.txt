Почему старый файл не работал:
iPhone открыл HTML во встроенном предпросмотре Files/ChatGPT. Он не запускает JavaScript.

Как запустить эту версию:
1. Распакуйте архив.
2. Загрузите папку на любой статический хостинг, например GitHub Pages, Netlify или Cloudflare Pages.
3. Откройте адрес в Safari.
4. Нажмите «Поделиться».
5. Выберите «На экран Домой».

После первого запуска приложение работает офлайн.
Результаты хранятся только на вашем iPhone в localStorage.
